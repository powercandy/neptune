## Canvas和SVG的区别

| Canvas | SVG |
| -- | -- |
| 较新，Apple私有技术发展而来 | 历史悠久，2003年成为W3C标准 |
| 像素，只能脚本驱动，动态渲染和大数据量绘制 | 矢量，XML，CSS，元素操作；适合静态图标展示，高保真文档查看和打印的应用场景 |
| 功能简单，2D绘图 | 功能丰富，各种图形、滤镜、动画等 |
| 主流浏览器，IE9+ | 主流浏览器，IE9+，其它SVG阅读器 |
